# SKUSavvy Inventory Intelligence Dashboard

Static GitHub Pages dashboard generated by GitHub Actions + Python.

## Files to upload / replace

Upload these files and folders to the repository root:

```text
index.html
scripts/generate_data.py
.github/workflows/update-dashboard.yml
README.md
.gitignore
```

## GitHub setup

1. Go to **Settings → Secrets and variables → Actions**.
2. Add a repository secret:
   - Name: `SKUSAVVY_TOKEN`
   - Value: your SKUSavvy API token
3. Go to **Settings → Pages**.
4. Select:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
5. Go to **Actions → Update SKUSavvy Dashboard Data → Run workflow**.

Your site URL should be:

```text
https://arojas-company.github.io/corro_skusavvy/
```

## Refresh behavior

- Automatic daily update: 6:00 AM UTC.
- Manual update: click **Update now** in the dashboard, then click **Run workflow** in GitHub Actions.
- After the workflow finishes, click **Reload data** in the dashboard.

```text

```

The script tries several SKUSavvy GraphQL warehouse-inventory query shapes. If SKUSavvy rejects them, the dashboard will not show false zeroes; it will use total inventory as a fallback and generate:

```text
data/schema-debug.json
```

Share that file if the warehouse numbers still need exact mapping.
